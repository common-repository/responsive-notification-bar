Fast loaded responsive notification bar

== Description ==
Responsive notification bar is a lightweight responsive and well designed plugin which you can use in your site to display some message, with a `call to action ` button to interact with visitor. 

It's very simple, only 5kb in size and well coded. Just get this plugin and start using it. 

Demo of this plugin --> http://plugins.wpflaty.com/responsive-notification-bar/

== Installation ==

1. Upload this plugin in your server ` /wp-content/plugins/ `
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If above two steps goes well then you will see a menu option with a pretty icon name ` RNB settings ` , going there you can customize this plugin.
